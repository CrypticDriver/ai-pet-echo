## 技术实现建议

### 对话选择逻辑

```python
def select_dialogue(scene, pet_mood, time_since_last_visit):
    # 根据场景、情绪、距离上次访问时间选择对话
    
    if scene == "greeting":
        hour = get_current_hour()
        if 6 <= hour < 12:
            variants = morning_greetings
        elif 18 <= hour < 24:
            variants = evening_greetings
        
        # 根据情绪调整
        if pet_mood == "happy":
            return random.choice([v for v in variants if v.tone == "cheerful"])
        elif pet_mood == "sad":
            return random.choice([v for v in variants if v.tone == "gentle"])
    
    # 特殊情况：长时间未见
    if time_since_last_visit > 7:
        return random.choice(long_absence_dialogues)
    elif time_since_last_visit > 3:
        return random.choice(medium_absence_dialogues)
```

### 变量替换

```
{X} → 天数/次数（需要从数据库读取）
[名字] → Pet的名字（用户设定）
```
