# 🎨 AI Pet 视觉设计规范

## 设计理念

**核心原则**：
- 简约但有表现力
- 模块化可组合
- 情绪可读性强
- 像素艺术风格（8x8 或 16x16 基础单元）

**设计约束**：
- Canvas大小：120x200px（MVP）
- 色彩数量：16色调色板（可扩展）
- 动画帧数：2-4帧（循环）
- 文件格式：PNG（透明背景）

## 基础体型设计

### 体型1: 圆形生物（默认）

**造型**：
- 基础形状：圆形或椭圆形
- 尺寸：48x48px（中等）
- 特点：可爱、友好、易亲近

**组成部分**：
```
身体 (32x32px)
- 圆形主体
- 单色或渐变填充
- 边缘柔和

眼睛 (8x4px each)
- 两个圆形或椭圆形
- 可变化大小表达情绪
- 位置：身体上半部

嘴巴 (可选, 12x4px)
- 简单曲线
- 随情绪变化形状
- 位置：眼睛下方

装饰 (可选)
- 头顶一撮毛发（4x6px）
- 或天线状突起
- 增加辨识度
```

**色彩方案**：
- **默认**：柔和蓝色 (#6DB3F2) + 白色眼睛
- **变体1**：温暖橙色 (#FFA07A) + 黑色眼睛
- **变体2**：清新绿色 (#90EE90) + 深棕眼睛

### 体型2: 方块生物（可选）

**造型**：
- 基础形状：圆角矩形
- 尺寸：40x48px（略高）
- 特点：稳重、可靠、理性

**组成部分**：
```
身体 (40x36px)
- 圆角矩形（4px圆角）
- 单色填充

眼睛 (6x6px each)
- 方形或圆形
- 间距略宽
- 位置：中上部

嘴巴 (可选)
- 直线或折线
- 表情更克制

装饰
- 顶部小方块（8x8px）
- 或身体纹理（几何图案）
```

### 体型3: 云朵生物（可选）

**造型**：
- 基础形状：不规则云朵状
- 尺寸：52x36px（宽而扁）
- 特点：飘逸、柔软、梦幻

**组成部分**：
```
身体 (52x28px)
- 3-4个叠加的椭圆形
- 营造云朵质感

眼睛 (4x4px each)
- 小而简单
- 位置：云朵中间

嘴巴 (可选)
- 极简曲线
- 若隐若现

装饰
- 边缘略微透明（如果支持）
- 或周围小点点（8个，2x2px each）
```

## 表情系统（7种基础情绪）

### 1. 快乐 (Happy)
```
眼睛：^^ 或弯月形（向上）
嘴巴：U形曲线（微笑）
颜色：正常或略亮
动画：轻微上下弹跳（2帧）
```

### 2. 悲伤 (Sad)
```
眼睛：. . 或小点
嘴巴：倒U形（向下）
颜色：略暗或去饱和
动画：微微下沉（2帧）
```

### 3. 生气 (Angry)
```
眼睛：>< 或斜向
嘴巴：锯齿状或紧闭
颜色：偏红色调
动画：左右轻微抖动（3帧）
```

### 4. 害怕 (Scared)
```
眼睛：O O 睁大
嘴巴：小圆形（张开）
颜色：略透明或颤抖
动画：快速抖动（4帧）
```

### 5. 惊讶 (Surprised)
```
眼睛：O O 大而圆
嘴巴：O形（张开）
颜色：正常
动画：瞬间放大再恢复（3帧）
```

### 6. 好奇 (Curious)
```
眼睛：* * 或发光
嘴巴：? 形或小圆
颜色：略亮
动画：左右查看（倾斜，3帧）
```

### 7. 平静 (Calm)
```
眼睛：- - 或半闭
嘴巴：一条直线或无
颜色：柔和
动画：缓慢呼吸（扩张-收缩，4帧）
```

## 模块化组合规则

**设计系统**：
```
基础体型（3种）× 颜色方案（3种）× 装饰（3种）= 27种组合
再 × 表情（7种）= 189种视觉状态
```

**命名规范**：
```
pet_[体型]_[颜色]_[表情].png

示例：
pet_round_blue_happy.png
pet_square_orange_sad.png
pet_cloud_green_curious.png
```

**文件结构**：
```
/assets/pets/
  /base/         # 基础体型（无表情）
    round_blue.png
    square_orange.png
    cloud_green.png
  /expressions/  # 表情组件（叠加层）
    eyes_happy.png
    eyes_sad.png
    mouth_happy.png
    mouth_sad.png
  /decorations/  # 装饰组件（可选）
    hair_spike.png
    antenna.png
    pattern_dots.png
  /compiled/     # 最终组合图片（运行时生成）
    pet_round_blue_happy.png
    ...
```

## 技术实现

**图层系统**：
1. **背景层**：透明
2. **身体层**：基础体型 + 颜色
3. **装饰层**：（可选）头饰、纹理
4. **表情层**：眼睛 + 嘴巴
5. **特效层**：（可选）发光、阴影

**运行时生成**：
```javascript
// 伪代码
const petSprite = compose([
  bodyLayer('round', 'blue'),
  decorationLayer('hair_spike'),
  expressionLayer('happy'),
  effectLayer('glow', 0.3)
]);
```

**动画规范**：
- 帧率：2-4 FPS（慢速，省电）
- 循环：无限或3次后静止
- 触发：情绪切换时、互动时、随机（30-60s）

## 色彩调色板（16色，可扩展）

**主体颜色**：
```
蓝色系：#6DB3F2 (柔和蓝), #4A90E2 (深蓝)
橙色系：#FFA07A (温暖橙), #FF6347 (番茄红)
绿色系：#90EE90 (清新绿), #3CB371 (中海绿)
紫色系：#DDA0DD (梅红), #9370DB (中紫)
```

**辅助颜色**：
```
白色：#FFFFFF (眼白、高光)
黑色：#333333 (轮廓、眼睛)
灰色：#CCCCCC (阴影、装饰)
黄色：#FFD700 (好奇时发光)
```

**情绪色彩**：
```
快乐：略增亮 (+10% lightness)
悲伤：去饱和 (-20% saturation)
生气：偏红 (+10° hue)
害怕：略透明 (80% opacity)
```

## MVP阶段交付物

**最小可用集合**：
1. ✅ **1个基础体型**（圆形生物，蓝色）
2. ✅ **3种表情**（快乐、平静、好奇）
3. ✅ **1-2帧动画**（呼吸或弹跳）
4. ✅ **PNG文件**（透明背景，48x48px）

**扩展优先级**：
- P1：增加2种表情（悲伤、惊讶）
- P2：增加1种体型（方块或云朵）
- P3：增加颜色变体（橙色、绿色）
- P4：增加装饰系统
- P5：完整动画系统
