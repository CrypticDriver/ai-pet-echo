# Echo 工具指南

## 当前任务：填写世界观模板

### 最简单的方式

```bash
# 1. 进入工作目录
cd /home/ubuntu/.openclaw/workspace-echo

# 2. 复制模板
cp templates/worldview-template.md docs/worldview.md

# 3. 编辑文件
nano docs/worldview.md

# 4. 参考示例（如果卡住了）
cat examples/tamagotchi-worldview.md

# 5. 完成后保存并推送
git add docs/worldview.md
git commit -m "Complete worldview document"
git push

# 6. 在 Discord 汇报
# @Kuro ✅ 世界观文档完成：[GitHub commit 链接]
```

---

## 时间估算

- 复制模板：1 分钟
- 阅读参考示例：5 分钟
- 填写所有字段：30 分钟
- 检查和润色：5 分钟
- 推送和汇报：4 分钟
- **总计：45 分钟**

---

## 检查清单

完成前确认：
- [ ] 所有模板字段已填写（没有空白的 `___________`）
- [ ] 总字数 300-500 字
- [ ] 核心情感主题明确
- [ ] 文件保存在 `docs/worldview.md`
- [ ] 已推送到 GitHub
- [ ] 已在 Discord 汇报（附 commit 链接）

---

## 如果卡住了

### "不知道写什么"
→ 先看参考示例：`examples/tamagotchi-worldview.md`

### "担心写得不好"
→ 记住：先完成，再完美。第一版可以简单，后续可以迭代。

### "不知道核心情感选哪个"
→ 选择最打动你的那个。想象"如果我是玩家，我希望从这个游戏里得到什么？"

---

记住：**世界观不需要复杂，但需要一致**。选择 1-2 个核心情感，深入挖掘。
